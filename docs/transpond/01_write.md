# 写作技巧类收录

1. [阮一峰 ——《科技爱好者周刊（第 288 期）：技术写作的首要诀窍）](https://www.ruanyifeng.com/blog/2024/01/weekly-issue-288.html)

3. [阮一峰 —— 《与王建硕的对话》](https://www.ruanyifeng.com/blog/2010/04/talk_with_wangjianshuo.html)
