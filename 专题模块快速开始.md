# 专题模块快速开始

只需 3 步即可添加新专题！

## 第 1 步：创建目录和 index.md

```bash
# 创建专题目录
mkdir -p content/topics/your-topic-name

# 创建配置文件
cat > content/topics/your-topic-name/index.md << 'EOF'
---
title: '你的专题标题'
description: '专题简短描述'
---

## 专题简介

这里可以写详细介绍...（可选）
EOF
```

## 第 2 步：添加文章

```bash
# 创建文章（会自动扫描并显示）
cat > content/topics/your-topic-name/first-article.md << 'EOF'
---
title: '第一篇文章'
description: '文章描述'
date: 2025-01-15
---

# 第一篇文章

文章内容...
EOF
```

## 第 3 步：构建和预览

```bash
npm run build
# 或
npm run dev
```

就这么简单！

## 排序文章

文章会自动按以下规则排序：

### 方式 1：使用 order（推荐用于手动排序）

```markdown
---
title: '文章标题'
order: 1  # 数字越小越靠前
---
```

### 方式 2：使用 date（推荐用于时间序列）

```markdown
---
title: '文章标题'
date: 2025-01-15  # 日期越新越靠前
---
```

### 方式 3：文件名排序（自动）

如果不设置 `order` 和 `date`，会按文件名字母顺序排列。

## 示例专题结构

```
content/topics/
├── react-best-practices/
│   ├── index.md          # 专题配置
│   ├── hooks.md          # 文章 1
│   └── performance.md    # 文章 2
└── system-prompt/
    ├── index.md          # 专题配置
    └── codex-cli-zh.md   # 文章 1
```

## 访问专题

- 专题列表：`/topics`
- 专题详情：`/topics/your-topic-name`
- 文章详情：`/topics/your-topic-name/first-article`
